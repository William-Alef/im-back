# Requisitos do exercício:

## 1. Organizando ícones nas redes sociais

Imagine que você está trabalhando em um projeto pessoal onde deseja incluir links para suas redes sociais com ícones correspondentes. Você já tem um link para o GitHub com um ícone, mas precisa adicionar outros dois: **LinkedIn** e **Twitch**. Utilize o trecho de código HTML fornecido como ponto de partida para incluir os links e ícones para essas duas redes adicionais.

## 2. Centralizando ícones e textos nos botões (usando Flexbox)

Imagine que você está trabalhando em um projeto de site onde precisa alinhar ícones de redes sociais com seus respectivos textos dentro de botões. O desafio é utilizar **Flexbox** para centralizar tanto o ícone quanto o texto dentro dos botões.

1.  Acesse o arquivo `style.css` e encontre a classe `.apresentacao__links__link`.
2.  Adicione as propriedades `display: flex`, `justify-content: center` e `gap: 16px` nesta classe, respeitando a ordem e posição corretas entre as outras propriedades existentes.



## 3. Espaçamento entre ícones e textos (Propriedade 'gap')

Neste desafio, você precisa ajustar o espaçamento entre os ícones e os textos dentro dos botões de redes sociais.

1.  Consulte o Figma do projeto para identificar o valor exato do espaçamento em pixels entre o ícone e o texto.
2.  Volte ao arquivo `style.css` e adicione a propriedade **`gap`** com o valor encontrado no Figma à classe `.apresentacao__links__link`.

Certifique-se de que o `gap` esteja posicionado corretamente dentro da declaração da classe.

## 4. Personalizando o efeito Hover (Borda Azul)

Você está desenvolvendo uma página de portfólio e quer criar um efeito visual atraente. O objetivo é fazer com que, ao passar o mouse sobre os botões, eles se destaquem com uma borda azul.

1.  Utilize a classe `.apresentacao__links__link` já existente no seu código CSS.
2.  Seu desafio é adicionar o pseudo-seletor `:hover` a essa classe, fazendo com que a borda do botão mude para a cor azul (`#0000FF`) quando o cursor passar sobre ele.

## 5. Alterando o cursor e o fundo dos botões no Hover

Na sua página de portfólio, você deseja duas alterações ao passar o mouse sobre um botão:

1.  O cursor deve se transformar em uma mão com o indicador levantado (`cursor: 'pointer'`).
2.  O fundo do botão deve mudar para um tom de preto levemente mais claro (`#272727`).

Seu desafio é editar a classe **`.apresentacao__links__link:hover`** no seu arquivo CSS para aplicar essas duas alterações.
