# Requisitos do exercício

## 1. Adaptando imagens para diferentes tamanhos de tela

Nesta atividade, você irá garantir que as imagens do site se adaptem corretamente a diferentes tamanhos de tela.  

Abra os arquivos HTML do projeto e localize as imagens relevantes da página. Em cada uma delas, adicione a classe `apresentacao__imagem`.  

Em seguida, abra o arquivo `styles.css` e defina a propriedade `width` dessa classe para `50%`, fazendo com que as imagens ocupem metade da largura do elemento pai.  

Após aplicar a estilização, utilize o navegador para testar a responsividade da página em diferentes larguras de tela, como `720px`, verificando se as imagens se ajustam corretamente sem quebrar o layout.

---

## 2. Ajustando o layout para diferentes dispositivos

Nesta atividade, você irá ajustar o layout do site para garantir uma boa experiência em diferentes dispositivos.  

Utilize a ferramenta **Inspecionar** do navegador para simular diferentes tamanhos de tela, como telas de notebook, tablet e dispositivos móveis.  

Observe como o layout se comporta em telas menores e faça os ajustes necessários para que a página continue organizada e visualmente agradável. Isso pode envolver alterações em propriedades como `margin`, `padding` ou até mesmo a reorganização dos elementos da página.  

Após os ajustes, teste novamente o layout em diferentes resoluções para validar as melhorias realizadas.

---

## 3. Responsividade do conteúdo de texto

O próximo passo é melhorar a responsividade do conteúdo textual do site.  

Abra o arquivo `styles.css` e localize a classe `.apresentacao__conteudo`. Caso ela esteja utilizando uma largura fixa, substitua por uma largura relativa.  

Defina a propriedade `width` como `50%` e observe como o texto se comporta ao alterar o tamanho da tela no navegador.  

Teste em diferentes resoluções para garantir que o conteúdo permaneça legível e bem distribuído, reforçando a importância da responsividade para uma boa experiência do usuário.

---

## 4. Melhorando a responsividade dos links de navegação

Nesta atividade final, você irá ajustar a responsividade dos links de navegação do site.  

No arquivo `styles.css`, localize a classe `.apresentacao__links__navegacao` e altere a propriedade `width` para um valor relativo, como `50%`.  

Após a modificação, teste o comportamento dos links em diferentes tamanhos de tela utilizando a ferramenta de inspeção do navegador. Observe se os links continuam acessíveis, organizados e fáceis de interagir.  

Caso necessário, faça ajustes adicionais para garantir uma navegação fluida e consistente em diferentes dispositivos.
