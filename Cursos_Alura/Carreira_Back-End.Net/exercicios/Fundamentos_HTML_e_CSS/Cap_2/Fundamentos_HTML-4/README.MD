# Requisitos do exercício

## 1. Espaçamento e tamanho dos botões

O desafio é ajustar o espaçamento e o tamanho dos botões da página.  

Após alinhar os botões horizontalmente utilizando **Flexbox**, você deve garantir que eles tenham um tamanho adequado e um espaçamento consistente entre si, conforme o design do projeto.  

Altere o CSS para:
- Ajustar o tamanho dos botões  
- Adicionar margens para criar o espaçamento correto entre eles  

---

## 2. Melhorando a visibilidade dos botões

Nesta atividade, o foco é melhorar a **acessibilidade e a visibilidade** dos botões.  

Foi identificado que os botões estavam difíceis de enxergar, portanto você deve:
- Aumentar o tamanho da fonte  
- Alterar as cores para melhorar o contraste  
- Adicionar efeitos visuais como borda ou sombra  

Utilize CSS para tornar os botões mais visíveis, atrativos e acessíveis ao usuário.

---

## 3. Criando uma classe comum para botões

Nesta etapa, você irá criar uma **classe CSS comum** para os botões "Instagram" e "GitHub" no HTML.  

Essa classe deve centralizar os estilos compartilhados entre os botões, incluindo:
- Cor de fundo  
- Tamanho do texto  
- Arredondamento dos vértices  

O objetivo é manter o código organizado, reutilizável e alinhado ao design do Figma.

---

## 4. Ajustando o espaçamento interno dos botões

O objetivo desta atividade é ajustar o **espaçamento interno** dos botões.  

Utilize a propriedade `padding` no CSS para definir o espaço entre o texto do botão e suas bordas, garantindo:
- Conforto visual  
- Melhor legibilidade  
- Consistência com o layout definido no Figma  

---

## 5. Importando e aplicando a fonte correta

Nesta atividade, você irá ajustar a tipografia dos botões.  

Importe a fonte **Montserrat** com peso **600** do Google Fonts e aplique-a aos botões utilizando a propriedade `font-family`.  

Além disso:
- Remova o sublinhado padrão dos links  
- Ajuste a cor do texto  

O objetivo é fazer com que o texto dos botões corresponda exatamente ao design definido no Figma.
