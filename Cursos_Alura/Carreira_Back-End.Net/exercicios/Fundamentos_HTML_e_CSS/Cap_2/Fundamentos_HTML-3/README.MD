# Requisitos do exercício

## 1. Espaçamento e margens com Flexbox

O objetivo é ajustar o espaçamento entre a seção de texto e a imagem para criar uma aparência semelhante ao design do Figma.  

Utilize a propriedade `justify-content: space-between;` em **Flexbox** para espaçar os elementos e adicione margens para evitar que eles fiquem colados nas bordas da tela.

---

## 2. Definindo o tamanho da seção de texto

Nesta atividade, você irá ajustar o tamanho de uma seção de texto para que corresponda às especificações do design no Figma.  

Adicione uma nova classe chamada `apresentacao__conteudo` à tag `<section>` no HTML e, em seguida, defina a largura dessa seção no CSS usando a propriedade `width`.  

O valor específico da largura deve ser retirado diretamente do projeto no Figma.

---

## 3. Ajustando tamanhos de títulos e textos

O objetivo é modificar o tamanho dos textos de títulos e parágrafos para que correspondam ao design no Figma.  

Crie classes específicas para o título (`apresentacao__conteudo__titulo`) e para o texto (`apresentacao__conteudo__texto`) no HTML.  

Depois, no CSS, utilize a propriedade `font-size` para definir os tamanhos de fonte dessas classes, com os valores obtidos no Figma.

---

## 4. Importando e aplicando fontes do Google Fonts

Nesta atividade, você irá importar as fontes **"Krona One"** e **"Montserrat"** do Google Fonts para o seu projeto web.  

Pesquise essas fontes no Google Fonts e utilize a opção de `@import` para incluí-las no arquivo CSS.  

Em seguida, aplique essas fontes aos elementos de texto específicos da página, como títulos e parágrafos, usando a propriedade `font-family`.

---

## 5. Personalizando fontes de títulos e textos

Após importar as fontes, personalize o estilo dos títulos e textos da página para que correspondam ao design no Figma.  

Modifique a propriedade `font-family` no CSS, aplicando:
- **"Krona One"** para os títulos  
- **"Montserrat"** para os textos  

O objetivo é garantir que o estilo visual da página esteja totalmente alinhado com o design proposto.
