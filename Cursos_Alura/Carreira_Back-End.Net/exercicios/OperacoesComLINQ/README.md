## Requisitos do exercício:

- Aplique filtros com o método Where para selecionar músicas por artista ou duração.
- Implemente a ordenação com OrderBy, OrderByDescending e ThenBy.
- Utilize as funções Take e Skip para paginar os resultados.
- Projete a coleção de músicas em uma coleção de artistas usando Select.
- Aplique o Distinct para eliminar duplicatas da lista de artistas.
- Extraia e transforme gêneros de CSV em uma lista simples utilizando SelectMany e Trim.
- Calcule estatísticas com funções de agregação como Count, Min, Max, Average e Sum.
- Materialize a coleção com ToList para garantir o processamento completo dos dados.
- Agrupe as músicas por artista usando GroupBy e conte os itens de cada grupo.
- Obtenha elementos específicos utilizando métodos como First e MaxBy.
- Verifique a existência de condições na coleção com Any ou métodos similares.
